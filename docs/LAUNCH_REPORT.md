| 竞品 | 数据库规模 | 起售价 | 核心优势 | 主要短板 |

```
✓ 透明定价 - 竞品大多需联系销售
✓ 中国支付支持 - 竞品均不支持支付宝/微信
```

## 二、AIAds 产品定位

### 一句话定位

> **AIAds 是中国工厂/企业投放全球中小 KOL 广告的首选平台，通过 AI 智能匹配和透明低价，让海外营销变得简单、便宜、有效。**

### 目标用户

**广告主**：
- 中国工厂/跨境电商/品牌方
- 年营收 1000 万 +
- 海外营销预算 10 万+/年
- 痛点：成本高、找不到 KOL、操作复杂

**KOL**：
- TikTok/YouTube/Instagram 平台
- 1 千 -5 万粉丝
- 有变现意愿但缺乏渠道
- 痛点：粉丝少接不到广告、收入不稳定

### 差异化优势

| 优势 | 说明 | 竞对情况 |
|------|------|---------|
| 小 KOL 聚焦 | 1 千 -5 万粉丝，避开头部竞争 | 竞品均聚焦 10 万 + |
| 中国供应链 | 深度整合供应商资源 | 竞品均无 |
| 成本优势 | 降低 70-90% (¥100-500/人) | 竞品$100-500/人 |
| 本地化服务 | 全中文、中国支付、本地客服 | 竞品均无 |
| 透明定价 | ¥999/月起 | 竞品需联系销售 |

## 三、产品规划

### MVP 功能范围（6 周）

**广告主端（P0）**：
- 注册登录、实名认证
- 充值支付（支付宝/微信/银行卡）
- 发布需求（预算、平台、受众、品类）
- KOL 搜索和筛选
- AI 推荐 KOL 列表
- 发起合作、内容审核、确认付款
- 基础数据看板

**KOL 端（P0）**：
- 注册登录、账号绑定
- 同步数据（粉丝数、互动率）
- 浏览任务广场、申请任务
- 发布内容、提交审核
- 收益明细、提现

**管理后台（P0）**：
- 用户管理、KOL 审核
- 内容审核、财务管理
- 数据报表

### 时间表

## 四、技术架构

### 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | React + TypeScript + MUI | 响应式 Web 应用 |
| **后端** | Node.js + Express | RESTful API |
| **数据库** | PostgreSQL (Supabase) | 主数据库 |
| **缓存** | Redis (Upstash) | 会话和缓存 |
| **部署** | Vercel + Railway | 自动 CI/CD |
| **文件存储** | Cloudflare R2 | 图片和视频 |

### 第三方服务

| 服务 | 用途 | 成本 |
|------|------|------|
| Supabase | 数据库 + Auth | 免费 500MB |
| TikTok API | KOL 数据同步 | 免费 |
| YouTube API | KOL 数据同步 | 免费 |
| Instagram Graph API | KOL 数据同步 | 免费 |
| Stripe | 国际支付 | 2.9% + $0.3 |
| 支付宝/微信 | 国内支付 | 0.6% |
| PayPal | KOL 提现 | 2.9% + $0.3 |

### 技术风险评估

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|----------|
| 社交平台 API 限制 | 中 | 高 | 提供手动上传选项 |
| 支付合规 | 中 | 高 | 使用持牌第三方 |
| KOL 数据造假 | 高 | 中 | 欺诈检测 + 人工审核 |

**评估结论**：技术可行，无重大不可控风险

### 主要风险

| 风险 | 概率 | 影响 | 建议 |
|------|------|------|------|
| 团队招募延迟 | 高 | 高 | 外包 + 兼职过渡 |
| API 审批时间长 | 中 | 高 | 提前申请，准备备选方案 |

### 核心建议

4. **本地化优势**：全中文界面、中国支付、本地客服
5. **代运营服务**：帮助工厂客户成功，提高留存
