## 1. 注册和认证

### 1.1 注册流程

#### 步骤 1：访问官网
打开浏览器，访问 [https://aiads.com](https://aiads.com)

#### 步骤 2：点击注册
在页面右上角点击「注册」按钮

#### 步骤 3：选择角色
选择「广告主」角色

#### 步骤 4：填写信息

| 字段 | 要求 | 示例 |
|------|------|------|
| 邮箱 | 有效邮箱地址 | company@example.com |
| 密码 | 8-20 位，含大小写字母和数字 | SecurePass123! |
| 手机号 | 中国大陆手机号 | 138****3800 |
| 验证码 | 邮箱/手机收到的 6 位验证码 | 123456 |

#### 步骤 5：同意协议
勾选「我已阅读并同意《用户服务协议》和《隐私政策》」

#### 步骤 6：完成注册
点击「注册」按钮，系统自动登录并跳转到广告主后台

### 1.2 实名认证

**为什么需要实名认证？**
- 发布广告活动必须完成认证
- 确保平台交易安全
- 符合法律法规要求

#### 认证类型

| 类型 | 适用对象 | 所需材料 |
|------|----------|----------|
| 企业认证 | 公司/企业 | 营业执照、法人身份证 |
| 个体工商户 | 个体经营者 | 营业执照、经营者身份证 |
| 个人认证 | 个人广告主 | 身份证 |

#### 企业认证流程

**步骤 1：进入认证页面**
- 登录后点击右上角用户名
- 选择「账户设置」
- 点击「实名认证」

**步骤 2：填写企业信息**

| 字段 | 说明 | 示例 |
|------|------|------|
| 公司名称 | 营业执照上的完整名称 | 某某科技有限公司 |
| 统一社会信用代码 | 18 位信用代码 | 91440300MA5XXXXX |
| 法人代表 | 营业执照上的法人姓名 | 张三 |
| 法人身份证号 | 18 位身份证号 | 440***********1234 |
| 联系人姓名 | 日常联系人 | 李四 |
| 联系人职位 | 联系人职位 | 市场经理 |
| 联系人电话 | 联系人手机号 | 138****3800 |
| 联系人邮箱 | 联系人邮箱 | contact@example.com |
| 公司官网 | 公司官方网站 | https://example.com |
| 所属行业 | 选择主要行业 | 电子产品 |
| 公司规模 | 员工人数范围 | 100-500 人 |

**步骤 3：上传证照**

| 材料 | 要求 | 格式 |
|------|------|------|
| 营业执照 | 清晰完整，四角齐全 | JPG/PNG，≤5MB |
| 法人身份证 | 正反面，清晰可辨 | JPG/PNG，≤5MB |
| 授权书（可选） | 如联系人非法人 | PDF/JPG，≤10MB |

**步骤 4：提交审核**
- 确认信息无误
- 点击「提交审核」

**步骤 5：等待审核**
- 审核时间：1-3 个工作日
- 审核结果通过邮件/短信通知
- 可在「认证进度」查看状态

#### 认证状态说明

| 状态 | 说明 | 操作 |
|------|------|------|
| 待提交 | 未填写认证信息 | 填写信息并提交 |
| 审核中 | 已提交，等待审核 | 等待审核结果 |
| 已通过 | 认证成功 | 可发布活动 |
| 已驳回 | 认证失败 | 查看原因，重新提交 |

### 1.3 账号安全

#### 设置两步验证（推荐）

1. 进入「账户设置」→「安全设置」
2. 点击「开启两步验证」
3. 使用验证器 App（如 Google Authenticator）扫描二维码
4. 输入 6 位验证码
5. 保存备用码（重要！）

#### 修改密码

1. 进入「账户设置」→「安全设置」
2. 点击「修改密码」
3. 输入当前密码
4. 输入新密码（8-20 位，含大小写字母和数字）
5. 确认新密码
6. 点击「保存」

#### 登录设备管理

1. 进入「账户设置」→「安全设置」
2. 查看「登录设备」列表
3. 可移除不认识的设备
4. 移除后该设备需要重新登录

### 2.2 充值流程

**步骤 1：进入充值页面**

- 选择「充值」

**步骤 2：选择充值金额**

| 预设金额 | 推荐 |
|----------|------|
| ¥500 | 新手体验 |
| ¥2,000 | ⭐ 推荐 |
| ¥5,000 | 常规投放 |

| 自定义 | 输入任意金额（≥¥100） |

**步骤 3：选择支付方式**
- 支付宝
- 微信支付
- 银联在线

**步骤 4：确认充值**
- 核对充值金额
- 点击「确认充值」

**步骤 5：完成支付**
- 跳转至支付平台
- 完成支付操作
- 自动返回 AIAds 平台

**步骤 6：查看余额**
- 充值成功后余额实时更新

- 点击「登记转账」

**步骤 4：等待入账**
- 财务确认到账
- 系统自动入账

## 3. 创建活动

### 3.1 活动创建流程

**入口：** 点击顶部导航「活动管理」→「创建活动」

#### 步骤 1：基本信息

| 字段 | 要求 | 说明 | 示例 |
|------|------|------|------|
| 活动标题 | 5-50 字符 | 简洁明了的活动名称 | 春季新品电子产品推广 |
| 活动描述 | 最多 500 字符 | 详细介绍活动内容和目标 | 推广我们最新推出的智能手表... |
| 活动目标 | 单选 | 选择投放目标 | 品牌曝光/用户互动/转化购买 |

**活动目标说明：**

| 目标 | 适用场景 | 优化指标 |
|------|----------|----------|
| 品牌曝光 | 新品发布、品牌宣传 | 曝光量、触达人数 |
| 用户互动 | 增加粉丝、提高活跃 | 点赞、评论、分享 |
| 转化购买 | 直接销售、引流 | 点击、购买转化 |

#### 步骤 2：目标受众

**人口统计：**

| 字段 | 选项 | 说明 |
|------|------|------|
| 年龄范围 | 13-17/18-24/25-34/35-44/45-54/55+ | 可多选 |
| 性别 | 全部/男性/女性 | 单选 |
| 语言 | 英语/西班牙语/中文等 | 可多选 |

**地理位置：**

| 级别 | 可选范围 |
|------|----------|
| 国家 | 全球 200+ 国家 |
| 省份/州 | 美国、澳大利亚等国家 |
| 城市 | 主要城市 |

**兴趣标签：**

| 类别 | 子类别 |
|------|--------|
| 美妆护肤 | 化妆、护肤、美发 |
| 时尚穿搭 | 服装、鞋包、配饰 |
| 科技数码 | 手机、电脑、智能设备 |
| 美食 | 烹饪、餐厅、零食 |
| 旅行 | 景点、酒店、户外 |
| 健身 | 运动、瑜伽、营养 |
| 母婴 | 育儿、玩具、孕产 |

**平台选择：**

| 平台 | 说明 |
|------|------|
| TikTok | 短视频平台，适合年轻用户 |
| YouTube | 长视频平台，适合深度内容 |

**KOL 要求：**

| 字段 | 范围 | 说明 |
|------|------|------|
| 粉丝量 | 1K-1000K+ | 设置最低和最高粉丝量 |
| 互动率 | 0%-20% | 最低互动率要求 |
| 视频平均播放量 | 1K-1M+ | 最低平均播放量 |

**预算类型：**

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 固定预算 | 设置总预算，系统智能分配 | 预算有限，追求最大效果 |
| 动态预算 | 根据效果实时调整 | 预算充足，灵活投放 |

**预算金额：**

| 建议 | 金额范围 | 可合作 KOL 数 |
|------|----------|--------------|
| 体验版 | ¥500-2,000 | 3-5 个 |
| 标准版 | ¥2,000-10,000 | 10-30 个 |
| 专业版 | ¥10,000-50,000 | 30-100 个 |
| 企业版 | ¥50,000+ | 100+ 个 |

**投放时间：**

| 字段 | 说明 |
|------|------|
| 开始日期 | 活动开始时间（可预约未来日期） |
| 结束日期 | 活动结束时间 |
| 投放周期 | 自动计算天数 |

**内容要求：**

| 字段 | 说明 | 示例 |
|------|------|------|
| 内容要求 | 对 KOL 创作内容的要求 | 需要展示产品外观和使用场景 |

**预览信息：**
- 基本信息
- 目标受众
- 预算设置
- 内容要求

**确认操作：**
1. 仔细检查所有信息
2. 勾选「我已阅读并同意《广告投放协议》」
3. 点击「提交活动」

**提交后：**
- 活动状态变为「草稿」或「待审核」
- 需要充值后才能启动
- 可在「活动管理」中查看

### 3.2 活动状态说明

| 状态 | 说明 | 可操作 |
|------|------|--------|
| 草稿 | 已创建未提交 | 编辑、删除、提交 |
| 待审核 | 已提交等待审核 | 查看进度 |

| 已暂停 | 活动暂时停止 | 恢复、编辑 |
| 已完成 | 活动结束 | 查看数据、下载报表 |

### 3.3 活动管理

#### 编辑活动

1. 进入「活动管理」
2. 找到目标活动
3. 点击「编辑」
4. 修改需要变更的信息
5. 保存

**注意：** 进行中的活动编辑后可能需要重新审核。

#### 暂停/恢复活动

**暂停活动：**
1. 进入活动详情页
2. 点击「暂停活动」
3. 确认操作

**恢复活动：**
1. 进入已暂停活动详情页
2. 点击「恢复活动」
3. 确认操作

#### 删除活动

1. 进入「活动管理」
2. 找到目标活动
3. 点击「删除」
4. 确认删除

**注意：** 只有「草稿」和「已完成」状态的活动可以删除。

### 4.3 AI 推荐

#### 使用场景

| 场景 | 说明 |
|------|------|
| 快速匹配 | 根据活动自动推荐 |
| 发现新品 | 发现优质新 KOL |
| 拓展投放 | 找到更多类似 KOL |

#### 推荐逻辑

AI 推荐基于以下因素：

1. **活动目标匹配度**（30%）
   - 目标受众重合度
   - 内容类别相关性

2. **KOL 质量评分**（25%）
   - 粉丝真实性
   - 互动质量
   - 内容质量

3. **历史表现**（25%）
   - 过往合作效果
   - 广告主评价
   - 任务完成率

4. **性价比**（20%）
   - 报价合理性
   - 预期 ROI

#### 推荐结果

| 字段 | 说明 |
|------|------|
| KOL 信息 | 头像、昵称、粉丝数 |
| 匹配度 | 0-100% 匹配分数 |
| 匹配理由 | 为什么推荐该 KOL |
| 预估效果 | 预期曝光、互动 |
| 报价 | 合作价格 |

### 4.4 发起合作

#### 合作流程

```
广告主发起 → KOL 接受 → 创建订单 → KOL 创作 → 提交审核 → 广告主确认 → 完成
```

#### 发起步骤

**步骤 1：选择 KOL**
- 在 KOL 列表或详情页
- 点击「发起合作」

**步骤 2：填写需求**

| 字段 | 说明 |
|------|------|
| 合作活动 | 选择关联的活动 |
| 内容要求 | 具体创作要求 |
| 期望发布时间 | 希望发布的日期 |
| 报价 | 愿意支付的价格 |
| 备注 | 其他说明 |

**步骤 3：确认提交**
- 预览合作需求
- 确认预算充足
- 点击「发送邀请」

**步骤 4：等待响应**
- KOL 会在 48 小时内响应
- 可在「消息中心」查看状态
- KOL 接受后自动创建订单

#### 合作状态

| 状态 | 说明 | 下一步 |
|------|------|--------|
| 已发送 | 邀请已发送给 KOL | 等待 KOL 响应 |
| 已接受 | KOL 接受了邀请 | 创建订单 |
| 已拒绝 | KOL 拒绝了邀请 | 可重新邀请或选择其他 KOL |
| 已过期 | 超过 48 小时未响应 | 可再次邀请 |

| 状态 | 说明 | 可操作 |
|------|------|--------|
| 待接受 | 已发送邀请，等待 KOL 接受 | 取消邀请 |
| 进行中 | KOL 已接受，创作中 | 查看进度 |
| 待审核 | KOL 已提交，等待审核 | 审核、驳回 |

### 6.3 订单操作

#### 审核作品

1. 进入订单详情
2. 查看 KOL 提交的内容
3. 检查是否符合要求
4. 选择：
   - **通过**：确认作品，进入结算
   - **驳回**：填写原因，KOL 修改后重新提交

#### 确认结算

1. 作品审核通过后
2. 系统自动结算
3. 金额从预算扣除
4. KOL 获得收益

#### 评价 KOL

1. 订单完成后
2. 进入订单详情
3. 点击「评价」
4. 填写：
   - 评分（1-5 星）
   - 评价内容
   - 是否推荐
5. 提交评价
