## 第一部分：广告主入门

### 1.1 注册账号

1. 访问 [AIAds 官网](https://aiads.com)
2. 点击右上角「注册」按钮
3. 选择「广告主」角色
4. 填写邮箱/手机号和密码
5. 完成邮箱/手机验证

### 1.2 实名认证

**企业用户需要完成实名认证才能发布活动：**

1. 登录后进入「账户设置」
2. 点击「实名认证」
3. 填写企业信息：
   - 公司名称
   - 营业执照号
   - 法人代表姓名
   - 联系人信息
4. 上传营业执照照片
5. 提交审核（1-3 个工作日）

### 1.3 充值账户

**支持多种支付方式：**

| 支付方式 | 适用用户 | 最低充值 |
|----------|----------|----------|
| 支付宝 | 中国大陆用户 | ¥100 |
| 微信支付 | 中国大陆用户 | ¥100 |
| 银行卡 | 企业用户 | ¥1000 |
| Stripe | 国际用户 | $50 |

**充值步骤：**
1. 进入「财务管理」→「充值」
2. 选择充值金额
3. 选择支付方式
4. 完成支付

### 1.4 创建广告活动

**四步轻松创建活动：**

#### 步骤 1：基本信息
- 填写活动标题（5-50 字符）
- 描述活动内容（最多 500 字符）
- 选择活动目标：
  - 品牌曝光
  - 用户互动
  - 转化购买

#### 步骤 2：目标受众
- 年龄范围（如 18-35 岁）
- 性别偏好
- 目标国家/地区
- 兴趣标签
- 目标平台（TikTok/YouTube/Instagram）
- KOL 粉丝量要求
- 最低互动率要求

#### 步骤 3：预算设置
- 选择预算类型（固定/动态）
- 设置预算金额（最低¥100）
- 选择开始和结束日期

#### 步骤 4：确认提交
- 预览所有信息
- 确认并提交

### 1.5 选择 KOL

**两种发现 KOL 的方式：**

#### 方式一：AI 智能推荐
1. 进入「KOL 发现」
2. 点击「AI 推荐」
3. 系统根据活动目标自动匹配
4. 查看推荐列表和匹配度

#### 方式二：自主搜索
1. 使用筛选条件：
   - 平台（TikTok/YouTube/Instagram）
   - 粉丝量范围
   - 互动率
   - 类别（美妆/科技/时尚等）
   - 国家/地区
2. 浏览 KOL 卡片
3. 查看详细信息

### 1.6 发起合作

1. 点击心仪 KOL 的「发起合作」
2. 填写合作需求：
   - 内容要求
   - 发布时间
   - 预算报价
3. 等待 KOL 接受
4. KOL 接受后自动创建订单

### 1.7 数据看板

**实时监控广告效果：**

| 指标 | 说明 |
|------|------|
| 曝光量 | 视频被观看次数 |
| 互动量 | 点赞 + 评论 + 分享总数 |
| CTR | 点击通过率 |
| 转化率 | 购买转化比例 |
| ROI | 投入产出比 |

**查看路径：**
- 进入「活动管理」
- 点击活动名称
- 切换到「数据分析」标签

## 第二部分：KOL 入门

### 2.1 注册账号

1. 访问 [AIAds 官网](https://aiads.com)
2. 点击右上角「注册」按钮
3. 选择「KOL」角色
4. 填写邮箱/手机号和密码
5. 完成邮箱/手机验证

### 2.2 账号认证

**认证后可获得更多任务和更高收益：**

1. 登录后进入「账号绑定」
2. 选择要绑定的平台：
   - TikTok
   - YouTube
   - Instagram
3. 授权 AIAds 访问账号数据
4. 系统自动同步粉丝数、互动数据
5. 完成认证

### 2.3 完善资料

**提高被选中的概率：**

1. 进入「个人设置」
2. 填写：
   - 个人简介
   - 擅长领域（美妆/科技/时尚等）
   - 报价设置
   - 标签（便于广告主发现）
3. 上传头像

### 2.4 浏览任务广场

**发现适合的合作机会：**

1. 进入「任务广场」
2. 使用筛选条件：
   - 平台
   - 预算范围
   - 类别
   - 截止日期
3. 浏览任务列表
4. 点击查看详情

### 2.5 申请任务

1. 找到感兴趣的任务
2. 点击「申请合作」
3. 填写申请说明（可选）
4. 等待广告主审核
5. 审核通过后接受任务

### 2.6 发布内容

**任务接受后的操作流程：**

1. 进入「我的任务」
2. 找到对应任务
3. 点击「提交作品」
4. 填写：
   - 视频链接
   - 内容描述
   - 发布截图
5. 提交审核

### 2.7 收益管理

**查看和管理收益：**

#### 收益明细
- 进入「收益管理」
- 查看每笔收入来源
- 状态说明：
  - 待结算：作品已提交，待广告主确认
  - 已结算：可提现余额
  - 已提现：已完成提现

#### 申请提现
1. 进入「收益管理」→「提现」
2. 选择提现金额（最低¥100）
3. 选择收款方式：
   - PayPal（国际）
   - 支付宝/微信（中国大陆）
   - 银行卡
4. 提交申请
5. 1-3 个工作日到账

## 第三部分：常见问题

### 广告主常见问题

#### Q1: 最低充值金额是多少？
**A:** 最低充值金额为¥100（或等值外币）。

#### Q2: 如何保证 KOL 粉丝质量？
**A:** AIAds 使用 AI 技术检测 KOL 粉丝质量，自动过滤虚假粉丝和异常互动。所有 KOL 都经过数据验证。

#### Q3: 活动发布后多久能找到 KOL？
**A:** 通常 24-48 小时内会有 KOL 申请。AI 推荐功能可以立即匹配符合条件的 KOL。

#### Q4: 如何确保 KOL 按时发布内容？
**A:** 
- 平台会提醒 KOL 截止日期
- 如 KOL 未按时发布，可申请退款
- 严重违约的 KOL 会被平台下架

#### Q5: 可以中途修改活动要求吗？
**A:** 可以。进入活动详情页，点击「编辑」修改。但已接受的订单需要与 KOL 协商。

#### Q6: 发票如何开具？
**A:** 进入「财务管理」→「发票管理」申请开具电子发票，支持增值税专用发票和普通发票。

#### Q7: 预算没用完可以退款吗？
**A:** 未使用的预算可以随时退回账户余额。已支付给 KOL 的费用不可退回。

### KOL 常见问题

#### Q1: 多少粉丝可以入驻？
**A:** 单个平台粉丝≥5000 即可入驻。多平台粉丝可累计计算。

#### Q8: 如何绑定多个社交账号？
**A:** 进入「账号绑定」，依次添加 TikTok、YouTube、Instagram 等账号。每个平台可绑定一个账号。

#### Q3: 收益多久可以提现？
**A:** 广告主确认作品后，收益进入可提现余额。随时可申请提现，1-3 个工作日到账。

#### Q4: 如何定价？
**A:** 系统会根据你的粉丝量、互动率、平台给出建议报价。也可自行设置。

| 粉丝量 | 建议报价（单视频） |
|--------|-------------------|
| 5K-10K | $50-150 |
| 10K-30K | $150-400 |
| 30K-50K | $400-800 |

#### Q5: 可以同时接多个任务吗？
**A:** 可以。但请确保能按时完成所有任务。建议新手从 1-2 个任务开始。

#### Q6: 作品被拒绝怎么办？
**A:** 
- 查看广告主的拒绝原因
- 根据反馈修改后重新提交
- 如有异议可联系平台客服仲裁

#### Q7: 如何获得更多合作机会？
**A:** 
- 完善个人资料
- 保持高通过率
- 按时交付优质内容
- 积累好评
