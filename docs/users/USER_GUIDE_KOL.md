## 1. 注册和认证

### 1.1 注册流程

#### 步骤 1：访问官网
打开浏览器，访问 [https://aiads.com](https://aiads.com)

#### 步骤 2：点击注册
在页面右上角点击「注册」按钮

#### 步骤 3：选择角色
选择「KOL」角色

#### 步骤 4：填写信息

| 字段 | 要求 | 示例 |
|------|------|------|
| 邮箱 | 有效邮箱地址 | creator@example.com |
| 密码 | 8-20 位，含大小写字母和数字 | SecurePass123! |
| 手机号 | 手机号（支持国际号码） | +1-xxx-xxx-xxxx |
| 验证码 | 邮箱/手机收到的 6 位验证码 | 123456 |

#### 步骤 5：同意协议
勾选「我已阅读并同意《用户服务协议》和《隐私政策》」

#### 步骤 6：完成注册
点击「注册」按钮，系统自动登录并跳转到 KOL 后台

### 1.2 入驻条件

#### 基本要求

| 平台 | 最低粉丝量 | 其他要求 |
|------|-----------|----------|
| TikTok | 5,000 | 近 30 天有发布内容 |
| YouTube | 5,000 | 近 90 天有发布内容 |
| Instagram | 5,000 | 近 30 天有发布内容 |

**多平台粉丝可累计计算**

#### 内容要求

- 原创内容占比≥80%
- 无违规记录
- 内容质量符合平台规范

#### 禁止入驻的情况

- 粉丝造假（使用刷粉服务）
- 互动造假（使用刷赞/刷评服务）
- 内容违规（色情、暴力、违法等）
- 账号被封禁或限制

### 1.3 实名认证

**为什么需要实名认证？**
- 提现必须完成认证
- 确保平台交易安全
- 符合法律法规要求

#### 认证类型

| 类型 | 适用对象 | 所需材料 |
|------|----------|----------|
| 个人认证 | 个人 KOL | 身份证/护照 |
| 企业认证 | MCN 机构/公司 | 营业执照、法人身份证 |

#### 个人认证流程

**步骤 1：进入认证页面**
- 登录后点击右上角头像
- 选择「账户设置」
- 点击「实名认证」

**步骤 2：填写个人信息**

| 字段 | 说明 | 示例 |
|------|------|------|
| 真实姓名 | 身份证上的姓名 | 张三 |
| 身份证号 | 18 位身份证号 | 440***********1234 |
| 国籍 | 选择国籍 | 中国/美国/英国等 |
| 出生日期 | 身份证上的出生日期 | 1990-01-01 |
| 性别 | 男/女 | 男 |

**步骤 3：上传证件**

| 材料 | 要求 | 格式 |
|------|------|------|
| 身份证正面 | 清晰完整，四角齐全 | JPG/PNG，≤5MB |
| 身份证反面 | 清晰完整，文字可辨 | JPG/PNG，≤5MB |
| 手持身份证 | 人脸和证件都清晰 | JPG/PNG，≤5MB |

**步骤 4：填写收款信息**

| 字段 | 说明 |
|------|------|
| 收款人姓名 | 必须与身份证一致 |
| 收款账号 | 支付宝/PayPal/银行账号 |
| 开户行 | 银行名称（银行卡提现需要） |

**步骤 5：提交审核**
- 确认信息无误
- 点击「提交审核」

**步骤 6：等待审核**
- 审核时间：1-3 个工作日
- 审核结果通过邮件/短信通知
- 可在「认证进度」查看状态

#### 认证状态说明

| 状态 | 说明 | 操作 |
|------|------|------|
| 待提交 | 未填写认证信息 | 填写信息并提交 |
| 审核中 | 已提交，等待审核 | 等待审核结果 |
| 已通过 | 认证成功 | 可提现 |
| 已驳回 | 认证失败 | 查看原因，重新提交 |

### 1.4 账号安全

#### 设置两步验证（强烈推荐）

1. 进入「账户设置」→「安全设置」
2. 点击「开启两步验证」
3. 使用验证器 App（如 Google Authenticator）扫描二维码
4. 输入 6 位验证码
5. 保存备用码（重要！请妥善保存）

#### 修改密码

1. 进入「账户设置」→「安全设置」
2. 点击「修改密码」
3. 输入当前密码
4. 输入新密码（8-20 位，含大小写字母和数字）
5. 确认新密码
6. 点击「保存」

#### 登录设备管理

1. 进入「账户设置」→「安全设置」
2. 查看「登录设备」列表
3. 可移除不认识的设备
4. 移除后该设备需要重新登录

## 2. 账号绑定

### 2.1 支持的平台

| 平台 | 功能 | 数据同步 |
|------|------|----------|
| TikTok | 绑定、同步、发布 | 实时 |
| YouTube | 绑定、同步 | 每小时 |
| Instagram | 绑定、同步 | 每小时 |

### 2.2 绑定流程

#### TikTok 绑定

**步骤 1：进入账号绑定页面**
- 点击顶部导航「账号绑定」
- 选择 TikTok 平台

**步骤 2：点击「绑定 TikTok 账号」**
- 系统跳转到 TikTok 授权页面

**步骤 3：授权 AIAds**
- 登录 TikTok（如未登录）
- 点击「授权」允许 AIAds 访问：
  - 基本信息（昵称、头像、粉丝数等）
  - 视频数据（播放量、点赞、评论等）
  - 发布权限（用于发布合作内容）

**步骤 4：完成绑定**
- 授权成功后自动返回 AIAds
- 显示绑定成功
- 开始同步数据

#### YouTube 绑定

**步骤 1：进入账号绑定页面**
- 点击顶部导航「账号绑定」
- 选择 YouTube 平台

**步骤 2：点击「绑定 YouTube 账号」**
- 系统跳转到 Google 授权页面

**步骤 3：授权 AIAds**
- 登录 Google 账号（如未登录）
- 选择要绑定的 YouTube 频道
- 点击「允许」授权 AIAds 访问：
  - 频道信息（名称、订阅数、视频数等）
  - 视频数据（播放量、点赞、评论等）

**步骤 4：完成绑定**
- 授权成功后自动返回 AIAds
- 显示绑定成功
- 开始同步数据

#### Instagram 绑定

**步骤 1：进入账号绑定页面**
- 点击顶部导航「账号绑定」
- 选择 Instagram 平台

**步骤 2：点击「绑定 Instagram 账号」**
- 系统跳转到 Instagram 授权页面

**步骤 3：授权 AIAds**
- 登录 Instagram（如未登录）
- 点击「授权」允许 AIAds 访问：
  - 基本信息（昵称、头像、粉丝数等）
  - 帖子数据（点赞、评论等）

**步骤 4：完成绑定**
- 授权成功后自动返回 AIAds
- 显示绑定成功
- 开始同步数据

### 2.3 数据同步

#### 同步的数据类型

| 数据类型 | TikTok | YouTube | Instagram |
|----------|--------|---------|-----------|
| 粉丝数 | ✅ | ✅ | ✅ |
| 关注数 | ✅ | ❌ | ✅ |
| 视频/帖子数 | ✅ | ✅ | ✅ |
| 总点赞数 | ✅ | ✅ | ✅ |
| 平均播放量 | ✅ | ✅ | ❌ |
| 平均点赞 | ✅ | ✅ | ✅ |
| 平均评论 | ✅ | ✅ | ✅ |
| 互动率 | ✅ | ✅ | ✅ |

#### 同步频率

| 类型 | 频率 |
|------|------|
| 实时同步 | 粉丝数变化超过 1000 时 |
| 每小时同步 | 正常情况 |
| 手动同步 | 用户主动点击同步按钮 |

#### 手动同步

1. 进入「账号绑定」
2. 找到已绑定的账号
3. 点击「同步」按钮
4. 等待同步完成（约 10-30 秒）

### 2.4 解绑账号

#### 解绑条件

- 没有进行中的合作任务
- 没有待结算的收益

#### 解绑步骤

1. 进入「账号绑定」
2. 找到要解绑的账号
3. 点击「解绑」
4. 确认解绑

**注意：** 解绑后该账号的数据将不再同步，但不影响历史合作记录。

### 2.5 多账号管理

#### 绑定多个账号

可以绑定同一平台的多个账号（需要特殊申请）：

1. 联系平台客服
2. 说明多账号需求
3. 提供账号列表
4. 客服审核后开通权限

#### 账号切换

1. 进入「账号绑定」
2. 选择要使用的账号
3. 点击「设为主账号」
4. 合作任务将关联到主账号

## 3. 任务广场

### 3.1 浏览任务

#### 入口
- 顶部导航「任务广场」

#### 任务列表

每个任务卡片显示：

| 信息 | 说明 |
|------|------|
| 活动标题 | 任务名称 |
| 广告主 | 公司名称（可能脱敏） |
| 平台 | 要求的发布平台 |
| 预算 | 任务报酬 |
| 粉丝要求 | 最低/最高粉丝量 |
| 截止日期 | 申请截止时间 |
| 标签 | 内容类别标签 |

### 3.2 筛选任务

#### 筛选条件

| 维度 | 选项 |
|------|------|
| 平台 | TikTok/YouTube/Instagram |
| 预算范围 | $50-100/100-300/300-500/500+ |
| 类别 | 美妆/时尚/科技/美食/旅行等 |
| 粉丝要求 | 根据自己粉丝量筛选 |
| 截止日期 | 今天/本周/本月 |

#### 排序方式

| 方式 | 说明 |
|------|------|
| 推荐排序 | AI 智能推荐（默认） |
| 最新优先 | 按发布时间倒序 |
| 预算最高 | 按预算从高到低 |
| 截止最近 | 按截止日期升序 |

### 3.3 任务详情

#### 基本信息

| 字段 | 说明 |
|------|------|
| 活动标题 | 任务名称 |
| 广告主 | 公司信息 |

#### 任务要求

| 字段 | 说明 |
|------|------|

| 必需标签 | 必须包含的 hashtag |
| 禁用内容 | 不允许出现的内容 |
| 参考视频 | 参考样例（如有） |

#### 报酬信息

| 字段 | 说明 |
|------|------|
| 基础报酬 | 完成任务的固定报酬 |
| 奖金 | 表现优异的额外奖励 |
| 结算方式 | 一次性/分期 |

#### 时间要求

| 字段 | 说明 |
|------|------|
| 申请截止 | 最晚申请时间 |
| 期望发布 | 希望发布的日期 |
| 最晚发布 | 必须发布的截止日期 |

#### 资格要求

| 字段 | 说明 |
|------|------|
| 粉丝量要求 | 最低/最高粉丝量 |
| 互动率要求 | 最低互动率 |
| 类别要求 | 内容类别 |
| 地区要求 | KOL 所在地区 |

### 3.4 申请任务

#### 申请条件

- 已完成实名认证
- 已绑定对应平台账号
- 粉丝量符合要求
- 账号状态正常

#### 申请步骤

**步骤 1：找到心仪任务**
- 在任务广场浏览或搜索
- 点击任务卡片查看详情

**步骤 2：点击「申请合作」**
- 在任务详情页点击「申请合作」按钮

**步骤 3：填写申请信息**

| 字段 | 说明 | 是否必填 |
|------|------|----------|
| 申请说明 | 自我介绍、创作想法 | 可选 |
| 期望报价 | 希望的报酬（可在预算范围内） | 可选 |
| 预计发布时间 | 计划发布的日期 | 可选 |
| 附件 | 作品集、案例等 | 可选 |

**步骤 4：提交申请**
- 预览申请信息
- 点击「提交申请」

**步骤 5：等待审核**
- 广告主会在 48 小时内审核
- 可在「我的任务」→「已申请」查看状态
- 审核结果通过邮件/站内信通知

#### 申请状态

| 状态 | 说明 | 下一步 |
|------|------|--------|
| 已提交 | 申请已发送给广告主 | 等待审核 |
| 已通过 | 广告主通过了申请 | 等待广告主创建订单 |
| 已拒绝 | 广告主拒绝了申请 | 可继续申请其他任务 |
| 已过期 | 超过申请截止时间 | 无法申请 |

### 3.5 提高申请成功率

#### 完善个人资料

- 上传专业头像
- 填写详细的个人简介
- 展示擅长领域
- 上传代表作品

#### 保持良好记录

- 按时完成任务
- 保证内容质量
- 积极沟通
- 积累好评

#### 精准申请

- 选择与自己风格匹配的任务
- 仔细阅读任务要求
- 针对性地填写申请说明
- 展示相关经验

## 4. 收益管理

### 4.1 收益构成

#### 收益类型

| 类型 | 说明 |
|------|------|
| 任务报酬 | 完成任务获得的基础报酬 |
| 绩效奖金 | 表现优异获得的额外奖励 |
| 推荐奖励 | 推荐其他 KOL 入驻的奖励 |
| 活动奖励 | 参与平台活动的奖励 |

#### 收益状态

| 状态 | 说明 | 可提现 |
|------|------|--------|
| 待结算 | 作品已提交，待广告主确认 | ❌ |
| 已结算 | 广告主已确认，可提现 | ✅ |
| 已提现 | 已完成提现 | ❌ |

### 4.2 收益明细

#### 查看收益

1. 进入「收益管理」
2. 查看收益列表

| 字段 | 说明 |
|------|------|
| 收益来源 | 任务名称/活动名称 |
| 类型 | 任务报酬/奖金/奖励 |
| 金额 | 收益金额 |
| 状态 | 待结算/已结算/已提现 |
| 时间 | 收益产生时间 |

#### 筛选收益

| 条件 | 说明 |
|------|------|
| 时间范围 | 按收益产生时间筛选 |
| 类型 | 按收益类型筛选 |
| 状态 | 按收益状态筛选 |
| 金额 | 按金额范围筛选 |

### 4.3 提现

#### 提现条件

- 已完成实名认证
- 已绑定收款账号
- 可提现余额≥¥100（或等值外币）
- 账号状态正常

#### 提现方式

| 方式 | 适用地区 | 最低提现 | 手续费 | 到账时间 |
|------|----------|----------|--------|----------|
| PayPal | 国际 | $50 | 2% | 1-3 工作日 |
| 支付宝 | 中国大陆 | ¥100 | 0% | 即时 |
| 微信支付 | 中国大陆 | ¥100 | 0% | 即时 |
| 银行转账 | 全球 | ¥500 | 1% | 3-5 工作日 |

#### 提现步骤

**步骤 1：进入提现页面**
- 点击顶部导航「收益管理」
- 选择「提现」

**步骤 2：选择提现金额**

| 选项 | 说明 |
|------|------|
| 全部提现 | 提现所有可提现余额 |
| 自定义 | 输入提现金额（≥最低限额） |

**步骤 3：选择收款方式**
- PayPal
- 支付宝
- 微信支付
- 银行转账

**步骤 4：填写收款信息**（首次使用需要填写）

| 方式 | 信息 |
|------|------|
| PayPal | PayPal 邮箱 |
| 支付宝 | 支付宝账号 |
| 微信支付 | 微信账号 |
| 银行转账 | 姓名、账号、开户行 |

**步骤 5：确认提现**
- 核对提现金额和收款信息
- 查看手续费
- 点击「确认提现」

**步骤 6：等待到账**
- 查看提现记录
- 预计时间内到账
- 到账后状态变为「已完成」

#### 提现记录

1. 进入「收益管理」→「提现记录」
2. 查看历史提现

| 字段 | 说明 |
|------|------|
| 提现单号 | 唯一提现编号 |
| 金额 | 提现金额 |
| 手续费 | 扣除的手续费 |
| 实收金额 | 实际到账金额 |
| 收款方式 | 提现方式 |
| 状态 | 处理中/已完成/已拒绝 |
| 时间 | 提现申请时间 |

#### 提现状态

| 状态 | 说明 |
|------|------|
| 处理中 | 平台正在处理 |
| 已打款 | 已转账到收款方 |
| 已完成 | 已确认到账 |
| 已拒绝 | 提现被拒绝（查看原因） |

### 4.4 税务说明

#### 个人所得税

- KOL 收益属于「劳务报酬所得」
- 平台依法代扣代缴个人所得税
- 可在「收益管理」→「纳税记录」查看

#### 纳税标准

| 收入金额 | 税率 |
|----------|------|
| ≤¥800 | 0% |
| ¥800-4000 | 20% - ¥800 |
| ¥4000-20000 | 20% × (1-20%) |
| ¥20000-50000 | 30% × (1-20%) - ¥2000 |
| >¥50000 | 40% × (1-20%) - ¥7000 |

#### 企业 KOL

- 以公司名义入驻的 KOL
- 平台开具增值税专用发票
- 公司自行申报纳税

## 5. 我的任务

### 5.1 任务状态

| 状态 | 说明 | 可操作 |
|------|------|--------|
| 待接受 | 广告主已创建订单，等待接受 | 接受/拒绝 |
| 进行中 | 已接受任务，创作中 | 提交作品 |
| 待审核 | 作品已提交，等待广告主审核 | 查看进度 |
| 待修改 | 作品被驳回，需要修改 | 修改后重新提交 |
| 已完成 | 广告主已确认，待结算 | 评价广告主 |
| 已取消 | 任务被取消 | 查看历史 |

### 5.2 任务列表

#### 查看任务

### 5.3 任务操作

#### 接受任务

1. 进入「待接受」任务详情
2. 仔细阅读任务要求
3. 确认可以按时完成
4. 点击「接受任务」

**注意：** 接受任务后请认真履约，频繁拒绝会影响信誉。

#### 拒绝任务

1. 进入「待接受」任务详情
2. 点击「拒绝任务」
3. 选择拒绝原因：
   - 时间冲突
   - 不符合风格
   - 报价不满意
   - 其他原因
4. 确认拒绝

#### 提交作品

1. 进入「进行中」任务详情
2. 点击「提交作品」
3. 填写：

| 字段 | 说明 | 是否必填 |
|------|------|----------|
| 视频链接 | 发布后的视频链接 | ✅ |
| 内容描述 | 视频内容介绍 | ✅ |
| 发布截图 | 发布后的截图 | ✅ |
| 备注 | 其他说明 | 可选 |

4. 点击「提交审核」

#### 修改作品

1. 进入「待修改」任务详情
2. 查看广告主的修改意见
3. 修改内容后重新发布
4. 更新视频链接和截图
5. 点击「重新提交」

#### 确认完成

1. 广告主审核通过后
2. 任务状态变为「已完成」
3. 等待收益结算

### 5.4 任务评价

## 7. 常见问题

### 7.1 账号相关

#### Q1: 多少粉丝可以入驻？
**A:** 单个平台粉丝≥5000 即可入驻。多平台粉丝可累计计算。

#### Q2: 可以绑定多个同平台账号吗？
**A:** 默认每个平台只能绑定一个账号。如有多账号需求，请联系客服申请。

#### Q3: 解绑账号后数据会清空吗？
**A:** 不会。解绑只停止数据同步，历史合作记录和收益数据都会保留。

#### Q4: 账号被封禁了怎么办？
**A:** 如社交账号被封禁，请及时联系平台客服，提供申诉材料，平台会协助处理。

### 7.2 任务相关

#### Q5: 如何获得更多任务机会？
**A:** 
- 完善个人资料，展示专业能力
- 保持高活跃度和内容质量
- 积累好评，提高信誉分
- 合理定价，提高竞争力
- 积极申请匹配的任务

#### Q6: 任务可以转给其他人吗？
**A:** 不可以。任务是基于您的账号和能力匹配的，必须由您本人完成。

#### Q7: 无法按时完成怎么办？
**A:** 请提前与广告主沟通，说明情况并申请延期。如广告主同意，可延长截止日期。

#### Q8: 作品被拒绝怎么办？
**A:** 
- 仔细查看广告主的拒绝原因
- 根据反馈修改后重新提交
- 如有异议可联系平台客服仲裁
- 严重分歧可申请平台介入

### 7.3 收益相关

#### Q9: 收益多久可以提现？
**A:** 广告主确认作品后，收益进入可提现余额。随时可申请提现，1-3 个工作日到账。

#### Q10: 提现被拒绝了怎么办？
**A:** 
- 查看拒绝原因
- 补充缺失的信息
- 重新提交提现申请

#### Q11: 收益如何纳税？
**A:** 平台依法代扣代缴个人所得税。可在「收益管理」→「纳税记录」查看明细。

#### Q12: 可以开发票吗？
**A:** 个人 KOL 由平台代扣税款，不单独开发票。企业 KOL 可开具增值税专用发票。

### 7.4 合作相关

#### Q13: 广告主不确认作品怎么办？
**A:** 
- 作品提交后广告主有 7 天审核期
- 如超时未审核，系统自动确认
- 如有争议可申请平台介入

#### Q14: 合作中可以修改要求吗？
**A:** 可以。双方协商一致后，广告主可修改任务要求。如差异较大，KOL 可拒绝。

#### Q15: 如何给广告主评价？
**A:** 任务完成后，进入任务详情页，点击「评价」即可给广告主评分和留言。
