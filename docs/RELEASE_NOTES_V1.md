# AIAds v1.0.0 发布说明

**版本**: 1.0.0
**发布日期**: 2026 年 3 月 24 日
**作者**: AIAds 产品团队

---

## 概述

AIAds v1.0.0 是我们平台的首个正式版本，标志着 AIAds 从产品规划阶段正式进入运营阶段。本版本实现了完整的 KOL 广告投放平台核心功能，包括广告主端、KOL 端和管理后台。

---

## 新功能

### 广告主端

#### 1. 账户管理
- **用户注册/登录**：支持邮箱/手机号注册，JWT 认证
- **实名认证**：企业认证流程，支持营业执照上传和审核
- **账户安全**：两步验证、密码管理、登录设备管理

#### 2. 充值支付
- **多种支付方式**：支付宝、微信支付、银联在线、Stripe（国际）
- **在线充值**：即时到账，支持预设金额和自定义金额
- **发票管理**：支持增值税专用发票和普通发票申请

#### 3. 活动管理
- **创建活动**：四步表单（基本信息、目标受众、预算设置、确认提交）
- **活动模板**：支持保存和复用活动模板
- **活动状态**：草稿、待审核、进行中、已暂停、已完成、已取消
- **活动编辑**：支持修改活动信息和预算

#### 4. KOL 发现
- **KOL 搜索**：多维度筛选（平台、粉丝量、互动率、类别、地区）
- **AI 推荐**：基于活动目标的智能 KOL 推荐
- **KOL 详情**：完整的数据展示（粉丝画像、历史合作、报价）
- **合作邀请**：直接向 KOL 发起合作邀请

#### 5. 订单管理
- **订单列表**：查看所有合作订单
- **订单状态**：待接受、进行中、待审核、已完成、已取消
- **作品审核**：审核 KOL 提交的作品，支持通过/驳回
- **评价系统**：对合作 KOL 进行评价

#### 6. 数据看板
- **核心指标**：预算使用、曝光量、互动量、ROI
- **趋势图表**：曝光趋势、互动趋势、消耗趋势
- **KOL 表现**：各 KOL 贡献数据排行
- **数据导出**：支持 Excel、CSV、PDF 格式导出

### KOL 端

#### 1. 账户管理
- **用户注册/登录**：支持邮箱/手机号注册
- **实名认证**：个人认证，支持身份证/护照验证
- **账号绑定**：绑定 TikTok、YouTube、Instagram 账号

#### 2. 账号同步
- **数据同步**：自动同步粉丝数、互动数据
- **多账号管理**：支持绑定多个平台账号
- **手动同步**：随时手动更新账号数据

#### 3. 任务广场
- **任务浏览**：查看所有可合作任务
- **任务筛选**：按平台、预算、类别筛选
- **任务详情**：完整的任务要求和报酬信息
- **任务申请**：在线申请感兴趣的任务

#### 4. 任务管理
- **我的任务**：查看所有任务（待接受/进行中/已完成）
- **任务操作**：接受/拒绝任务、提交作品、修改作品
- **任务状态**：实时跟踪任务进度

#### 5. 收益管理
- **收益统计**：可用余额、待结算金额、累计收益
- **收益明细**：每笔收益的来源和状态
- **提现功能**：支持 PayPal、支付宝、微信、银行转账
- **提现记录**：查看历史提现记录

#### 6. 个人设置
- **资料管理**：头像、简介、擅长领域
- **报价设置**：设置各平台报价和套餐价格
- **通知设置**：自定义通知渠道和类型

### 管理后台

#### 1. 用户管理
- **用户列表**：查看所有用户信息
- **用户详情**：完整的用户数据和行为记录
- **用户状态**：启用/禁用用户账号
- **权限管理**：子账号角色和权限配置

#### 2. KOL 管理
- **KOL 列表**：查看所有 KOL 信息
- **KOL 审核**：审核 KOL 认证申请
- **风险管理**：标记可疑 KOL，设置风险等级
- **数据监控**：监控 KOL 数据异常

#### 3. 活动管理
- **活动列表**：查看所有活动
- **活动审核**：审核广告主提交的活动
- **活动下架**：强制下架违规活动
- **数据查看**：查看活动详细数据

#### 4. 订单管理
- **订单列表**：查看所有订单
- **订单详情**：完整的订单信息
- **订单仲裁**：处理有争议的订单
- **数据统计**：订单转化率、完成率

#### 5. 财务管理
- **充值记录**：查看所有充值订单
- **提现审核**：审核 KOL 提现申请
- **发票管理**：查看发票申请和开具状态
- **财务概览**：平台收入和支出统计

#### 6. 数据报表
- **平台统计**：用户数、活动数、订单数、GMV
- **趋势分析**：各项指标的趋势图表
- **报表导出**：导出平台数据报表
- **操作日志**：管理员操作记录

## 技术架构

### 前端
- **框架**：React 19 + TypeScript
- **UI 库**：Material-UI (MUI) v7
- **状态管理**：Zustand
- **数据获取**：React Query v5
- **路由**：React Router v7
- **构建工具**：Vite

### 后端
- **运行时**：Node.js 20
- **框架**：Express
- **语言**：TypeScript
- **ORM**：Prisma
- **认证**：JWT (RS256)
- **验证**：Zod

### 数据库
- **主数据库**：PostgreSQL 15 (Supabase)
- **缓存**：Redis 7 (Upstash)
- **文件存储**：Cloudflare R2

### 部署
- **前端托管**：Vercel
- **后端托管**：Railway
- **DNS/CDN**：Cloudflare
- **CI/CD**：GitHub Actions

### 第三方集成
- **社交平台**：TikTok API、YouTube API、Instagram API
- **支付**：支付宝、微信支付、Stripe、PayPal
- **通讯**：SendGrid（邮件）、Twilio（短信）

## 升级指南

### 从测试版升级

如果您使用的是测试版本，请按以下步骤升级到正式版：

#### 1. 备份数据

```bash
# 备份数据库
pg_dump "$DATABASE_URL" > backup_$(date +%Y%m%d).sql

# 备份环境变量
railway variables export > vars_backup.json
```

#### 2. 更新代码

```bash
# 拉取最新代码
git pull origin main

# 安装依赖
npm install
```

#### 3. 数据库迁移

```bash
# 应用迁移
npx prisma migrate deploy

# 验证迁移
npx prisma db check
```

#### 4. 更新配置

```bash
# 检查环境变量
# 确保以下新变量已配置：
# - R2_ACCOUNT_ID
# - R2_ACCESS_KEY_ID
# - R2_SECRET_ACCESS_KEY
# - SENTRY_DSN
```

#### 5. 重启服务

```bash
# Railway 重启
railway restart

# 验证服务
curl https://api.aiads.com/health
```

### 从 v0.x 升级

v0.x 用户需要注意以下破坏性变更：

#### API 变更

| 旧端点 | 新端点 | 说明 |
|--------|--------|------|
| `/api/auth/register` | `/v1/auth/register` | 添加版本前缀 |
| `/api/campaigns` | `/v1/campaigns` | 添加版本前缀 |

#### 数据库变更

- `users` 表新增 `two_factor_enabled` 字段
- `campaigns` 表新增 `audit_status` 字段
- 新增 `audit_logs` 表
