## 五、实验方法论

### 5.1 实验设计原则

1. **单一变量**: 每次只测试一个变量
2. **随机分配**: 用户随机分配到实验组
3. **样本量充足**: 确保统计功效
4. **护栏指标**: 监控负面效应
5. **长期追踪**: 关注长期影响

### 5.2 统计方法

**假设检验**:
- 检验类型：双样本 t 检验
- 显著性水平：α = 0.05
- 检验效能：1-β = 0.80

**置信区间**:
- 置信水平：95%
- 计算方法：Bootstrap

**多重检验校正**:
- 方法：Bonferroni 校正
- 适用场景：同时检验多个指标

### 5.3 实验平台

**技术栈**:
- 实验分配：内部实验平台
- 数据统计：ClickHouse
- 显著性计算：Python + SciPy
- 可视化：Grafana

**实验流程**:
```
1. 提出假设 → 2. 设计实验 → 3. 确定样本量
       ↓
4. 开发实验 → 5. 灰度发布 → 6. 数据收集
       ↓
7. 统计分析 → 8. 得出结论 → 9. 决策发布
```

function assignExperiment(userId: string, experimentId: string): string {
