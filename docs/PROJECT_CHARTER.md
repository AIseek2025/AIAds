# AIAds 项目章程

## 项目愿景

打造全球领先的"AI×中国供应链×全球中小 KOL"广告投放平台，帮助中国工厂/企业以最低成本获取全球曝光，同时为全球中小 KOL 提供简单可靠的变现渠道。

## 核心价值主张

### 对广告主（中国工厂/企业）

- **低成本**：中小 KOL 广告成本仅为头部 KOL 的 1/10-1/100
- **高 ROI**：精准匹配，按效果付费
- **省心**：一站式管理，智能投放，自动优化
- **供应链优势**：深度整合中国供应商资源，提供选品 - 投放 - 转化闭环

### 对 KOL（全球中小博主）

- **稳定收入**：持续的任务来源，可预期的收入
- **低门槛**：几千粉丝即可入驻，无保证金
- **简单操作**：一键接单，发布即赚
- **公平透明**：清晰的定价和结算机制

### 平台价值

- **佣金收入**：10-20% 交易佣金
- **数据价值**：积累 KOL 和投放数据，优化匹配算法
- **生态壁垒**：双边网络效应，先入者优势

## 商业模式

### 收入来源

1. **交易佣金**：10-20%（主要收入）
2. **增值服务**：数据分析、内容优化建议
3. **广告收入**：平台内广告位（未来）

### 成本结构

1. **技术成本**：服务器、第三方 API
2. **运营成本**：KOL 审核、内容审核
3. **获客成本**：广告主和 KOL 获取
4. **人力成本**：研发、运营、客服

### 单位经济模型

| 指标 | 数值 |
|------|------|
| 平均客单价（广告主） | ¥50,000/年 |
| 平均 KOL 收益 | ¥5,000/年 |

| 单笔交易佣金 | ¥750 |
| 获客成本（广告主） | ¥2,000 |
| 获客成本（KOL） | ¥100 |
| 回本周期 | 3 个月 |

## 竞争壁垒

1. **中国供应链深度整合**：独家供应商资源，竞对无法复制
2. **中小 KOL 聚焦**：避开与头部平台的直接竞争
3. **AI 匹配算法**：数据积累形成算法壁垒
4. **双边网络效应**：广告主和 KOL 互相吸引
5. **先发优势**：快速占领市场，建立品牌认知
