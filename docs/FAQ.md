## 1. 账号与登录

### 1.1 如何登录系统？

**访问地址**: https://admin.aiads.com

**登录步骤**:
1. 打开浏览器访问系统地址
2. 输入管理员分配的邮箱和密码
3. 点击"登录"按钮
4. 如启用了 MFA，输入验证码
5. 登录成功

### 1.2 忘记密码怎么办？

**解决方法**:
1. 在登录页面点击"忘记密码"
2. 输入注册邮箱
3. 查收密码重置邮件
4. 点击邮件中的重置链接
5. 设置新密码
6. 使用新密码登录

**如未收到邮件**:
- 检查垃圾邮件箱
- 确认邮箱地址正确
- 联系超级管理员手动重置

### 1.3 MFA 验证码收不到怎么办？

**可能原因**:
- 手机信号不好
- 短信被拦截
- 验证码服务延迟

**解决方法**:
1. 检查手机信号和网络
2. 检查短信拦截记录
3. 等待 1-2 分钟后重试
4. 点击"重新发送"
5. 如仍收不到，联系超级管理员暂时关闭 MFA

### 1.4 账号被禁用怎么办？

**可能原因**:
- 违规操作
- 长时间未登录
- 安全风险

**解决方法**:
1. 联系直属主管说明情况
2. 主管审核后联系超级管理员
3. 超级管理员审核后解封账号

### 1.5 如何修改密码？

**修改步骤**:
1. 登录后点击右上角用户菜单
2. 选择"个人设置"
3. 点击"修改密码"
4. 输入当前密码
5. 输入新密码（需符合复杂度要求）
6. 确认新密码
7. 点击"保存"

**密码要求**:
- 至少 8 个字符
- 包含大写字母
- 包含小写字母
- 包含数字
- 包含特殊字符

### 1.6 如何启用 MFA？

**启用步骤**:
1. 登录后进入"个人设置"
2. 点击"启用 MFA"
3. 使用手机下载认证 APP（Google Authenticator/Microsoft Authenticator）
4. 扫描二维码或手动输入密钥
5. 输入 APP 生成的 6 位验证码
6. 保存备用验证码（重要！）
7. MFA 启用成功

### 1.7 更换手机后如何使用 MFA？

**解决方法**:
1. 使用备用验证码登录
2. 进入"个人设置"
3. 关闭 MFA
4. 重新启用 MFA 并绑定新手机

**如无备用验证码**:
- 联系超级管理员重置 MFA

### 1.8 Session 超时怎么办？

**超时原因**:
- 超过 8 小时未操作
- 系统强制登出

**解决方法**:
- 重新登录即可
- 建议启用"记住我"功能（7 天内免登录）

## 2. 权限与角色

### 2.1 为什么看不到某个功能菜单？

**可能原因**:
- 您的角色没有该功能的访问权限

**解决方法**:
1. 确认您需要该功能的原因
2. 向直属主管申请权限
3. 主管审批后由超级管理员配置

### 2.2 为什么操作按钮是灰色的？

**可能原因**:
- 您有查看权限但没有操作权限

**解决方法**:
- 申请相应操作权限

### 2.3 如何查看自己的权限？

**查看方法**:
1. 进入"个人设置"
2. 查看"我的权限"
3. 或联系超级管理员查询

### 2.4 权限配置后多久生效？

**生效时间**:
- 权限配置后立即生效
- 如未生效，退出重新登录

### 2.5 有哪些预定义角色？

| 角色 | 说明 |
|-----|------|
| **超级管理员** | 系统最高权限，可管理所有功能 |
| **管理员** | 日常运营管理，用户管理，内容审核 |
| **审核员** | KOL 审核、内容审核 |
| **财务** | 财务管理、提现审核 |
| **数据分析师** | 数据查看、报表导出 |

### 2.6 如何申请更多权限？

**申请流程**:
1. 向直属主管说明需要权限的原因
2. 主管审批
3. 超级管理员配置权限
4. 权限生效

## 3. 用户管理

### 3.1 如何搜索用户？

**搜索方法**:
1. 进入"用户管理"
2. 在搜索框输入用户名/邮箱/手机号
3. 支持模糊搜索
4. 点击"搜索"按钮

### 3.2 如何筛选用户？

**筛选条件**:
- 用户类型：广告主/KOL/普通用户
- 状态：正常/封禁
- 注册时间：自定义时间范围

### 3.3 如何封禁用户？

**封禁步骤**:
1. 在用户列表找到目标用户
2. 点击"封禁"按钮
3. 选择封禁原因
4. 选择封禁时长（临时/永久）
5. 填写封禁说明
6. 进行二次验证（MFA/短信）
7. 确认封禁

### 3.4 如何解封用户？

**解封步骤**:
1. 在用户列表找到已封禁的用户
2. 点击"解封"按钮
3. 填写解封说明
4. 确认解封

### 3.5 如何重置用户密码？

**重置步骤**:
1. 进入用户详情
2. 点击"重置密码"
3. 系统生成临时密码
4. 将临时密码发送给用户
5. 用户首次登录需修改密码

### 3.6 如何查看用户操作历史？

**查看方法**:
1. 进入用户详情
2. 点击"操作历史"标签
3. 查看用户的操作记录

### 3.7 删除用户和封禁用户有什么区别？

| 操作 | 效果 | 可恢复 |
|-----|------|--------|
| **封禁** | 用户无法登录，数据保留 | 可解封 |
| **删除** | 用户账号永久删除（软删除） | 不可恢复 |

**建议**: 优先使用封禁，删除操作需谨慎

## 4. KOL 管理

### 4.1 KOL 认证审核流程是什么？

**审核流程**:
1. KOL 提交认证申请
2. 系统自动初审（资料完整性）
3. 审核员人工审核
4. 审核通过→认证成功
5. 审核拒绝→通知 KOL 补充材料

### 4.2 KOL 审核时效要求是什么？

**时效要求**: 24 小时内完成审核

**查看待审核**:
- 数据总览待办事项
- KOL 管理→待审核

### 4.3 如何批量审核 KOL？

**批量审核步骤**:
1. 进入"KOL 管理"→"待审核"
2. 勾选多个 KOL
3. 点击"批量审核"按钮
4. 选择审核结果（通过/拒绝）
5. 填写审核意见
6. 确认提交

### 4.4 KOL 评级标准是什么？

| 评级 | 粉丝数 | 互动率 | 内容质量 | 合作履约率 |
|-----|-------|-------|---------|-----------|
| **S** | 5 万 + | 5%+ | 优秀 | 95%+ |
| **A** | 1 万 -5 万 | 3-5% | 良好 | 90%+ |
| **B** | 5 千 -1 万 | 2-3% | 一般 | 80%+ |
| **C** | 1 千 -5 千 | 1-2% | 合格 | 70%+ |

### 4.5 如何调整 KOL 评级？

**调整步骤**:
1. 进入 KOL 详情
2. 点击"调整评级"
3. 选择新评级
4. 填写调整原因
5. 确认提交

### 4.6 如何将 KOL 加入黑名单？

**加入黑名单步骤**:
1. 进入 KOL 详情
2. 点击"加入黑名单"
3. 选择加入原因
4. 填写说明
5. 确认提交

**黑名单效果**:
- KOL 无法接单
- KOL 不会在推荐列表中出现

### 4.7 KOL 数据如何同步？

**同步方式**:
- 自动同步：每天凌晨自动同步
- 手动同步：在 KOL 详情点击"同步数据"

**同步内容**:
- 粉丝数
- 关注数
- 作品数
- 互动数据

## 5. 活动与订单

### 5.1 活动状态有哪些？

| 状态 | 说明 |
|-----|------|
| **草稿** | 活动未提交审核 |
| **待审核** | 等待审核 |
| **已审核** | 审核通过，可上线 |
| **进行中** | 活动正在进行 |
| **已完成** | 活动结束 |
| **已暂停** | 活动暂停 |
| **已拒绝** | 审核拒绝 |

### 5.2 活动审核要点是什么？

**审核要点**:
- 活动内容是否合规
- 预算是否合理
- 目标 KOL 类型是否明确
- 活动时间是否合理

### 5.3 订单状态有哪些？

| 状态 | 说明 |
|-----|------|
| **待付款** | 广告主未付款 |
| **待接单** | KOL 未接单 |
| **进行中** | KOL 已接单，创作中 |
| **待审核** | KOL 提交内容，待审核 |
| **待发布** | 内容已审核，待发布 |
| **已完成** | 订单完成 |
| **已取消** | 订单已取消 |
| **纠纷中** | 存在纠纷 |

### 5.4 如何处理订单纠纷？

**处理步骤**:
1. 进入订单详情
2. 查看纠纷详情和双方陈述
3. 查看相关证据
4. 做出裁决
5. 填写裁决说明
6. 提交裁决结果

**裁决选项**:
- 支持广告主
- 支持 KOL
- 折中方案

### 5.5 如何导出订单数据？

**导出步骤**:
1. 进入"订单管理"
2. 设置筛选条件
3. 点击"导出"按钮
4. 选择导出格式（Excel/CSV）
5. 下载导出文件

## 6. 财务管理

### 6.1 提现审核流程是什么？

**审核流程**:
1. KOL 提交提现申请
2. 系统自动初审（余额检查、风控检查）
3. 财务管理员复审
4. 审核通过→打款→完成
5. 审核拒绝→退回→通知 KOL

### 6.2 提现审核时效要求是什么？

**时效要求**: 24 小时内完成审核

### 6.3 大额提现有什么特殊要求？

**大额定义**: 单笔超过¥10,000

**特殊要求**:
- 需要二次验证（MFA）
- 可能需要额外审核材料
- 审核时间可能延长

### 6.4 充值未到账怎么办？

**处理步骤**:
1. 检查充值状态
   - 待确认：等待财务确认
   - 已确认：充值已完成
2. 如状态为"待确认"，联系财务管理员
3. 提供充值凭证
4. 财务核实后确认充值

### 6.5 如何生成财务报表？

**生成步骤**:
1. 进入"财务管理"→"财务报表"
2. 选择报表类型（日报/周报/月报/自定义）
3. 选择时间范围
4. 点击"生成报表"
5. 查看或下载报表

### 6.6 如何调整用户余额？

**调整步骤**:
1. 进入用户详情
2. 点击"余额管理"
3. 输入调整金额（正数增加，负数减少）
4. 填写调整原因
5. 进行二次验证（MFA）
6. 确认调整

**注意**: 余额调整权限仅限超级管理员

## 7. 内容审核

### 7.1 内容审核流程是什么？

**审核流程**:
1. KOL 提交内容
2. 系统自动初审（敏感词检测）
3. 审核员人工审核
4. 审核通过→待发布
5. 审核拒绝→通知 KOL 修改

### 7.2 内容审核时效要求是什么？

**时效要求**: 2 小时内完成审核

### 7.3 内容审核要点是什么？

**审核要点**:
- 内容是否符合法律法规
- 内容是否清晰完整
- 是否符合品牌要求
- 是否为原创内容
- 是否有敏感词或违规内容

### 7.4 如何批量审核内容？

**批量审核步骤**:
1. 进入"内容审核"→"待审核"
2. 勾选多个内容
3. 点击"批量审核"按钮
4. 选择审核结果（通过/拒绝）
5. 填写审核意见
6. 确认提交

### 7.5 敏感词如何管理？

**管理步骤**:
1. 进入"内容审核"→"敏感词管理"
2. 查看当前敏感词列表
3. 添加/删除敏感词
4. 设置敏感级别

**敏感级别**:
- 高：直接拒绝
- 中：需要人工审核
- 低：仅提示

## 8. 系统使用

### 8.1 页面加载慢怎么办？

**可能原因**:
- 网络连接问题
- 浏览器缓存过多
- 系统负载高

**解决方法**:
1. 检查网络连接
2. 清除浏览器缓存
3. 尝试使用其他浏览器
4. 如持续缓慢，联系技术支持

### 8.2 系统提示错误怎么办？

**处理步骤**:
1. 记录错误信息和时间
2. 刷新页面重试
3. 如持续报错，联系技术支持
4. 提供错误截图和信息

### 8.3 数据不一致怎么办？

**处理步骤**:
1. 刷新页面
2. 清除缓存后重试
3. 如仍不一致，联系技术支持排查

### 8.4 如何导出大量数据？

**最佳实践**:
- 单次导出建议不超过 10 万条
- 使用筛选条件缩小数据范围
- 分批次导出
- 导出文件妥善保管

### 8.5 系统维护时间是什么时候？

**定期维护**: 每周日凌晨 02:00-06:00（UTC+8）

**维护通知**:
- 提前 3 天在系统公告通知
- 维护期间可能无法访问

### 8.6 支持哪些浏览器？

**推荐浏览器**:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

**最低要求**:
- 支持 JavaScript 的现代浏览器
- 启用 Cookie

### 8.7 系统支持移动端吗？

**支持情况**:
- 支持 iPad 及以上平板设备
- 手机浏览器可访问但体验不佳
- 推荐使用桌面浏览器
