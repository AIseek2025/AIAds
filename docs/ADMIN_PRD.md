## 1. 产品概述

### 1.1 产品定位

AIAds 管理后台是 AIAds 影响者营销平台的运营管理中枢，为平台运营人员、审核人员、财务人员和管理员提供统一的数字化运营管理界面。

**核心价值**:
- **高效运营**: 一站式管理用户、KOL、内容、订单和财务
- **智能审核**: AI 辅助审核，提高审核效率和准确性
- **数据驱动**: 实时数据看板，支持科学决策
- **安全可控**: 完善的权限体系和审计日志

### 1.2 目标用户

| 用户角色 | 描述 | 核心需求 | 使用频率 |
|---------|------|---------|---------|
| **超级管理员** | 系统最高权限持有者 (CTO/技术负责人) | 系统配置、管理员管理、权限管理、查看审计日志 | 每日 |
| **运营管理员** | 日常运营管理人员 (运营主管) | 用户管理、KOL 审核、内容审核、活动管理、订单管理 | 每日 |
| **审核管理员** | 内容/KOL 审核人员 (审核团队) | KOL 认证审核、内容审核、审核历史查看 | 每日 |
| **财务管理员** | 财务管理人员 (财务人员) | 充值管理、提现审核、交易记录、财务报表、发票管理 | 每日 |

### 1.3 核心价值

| 价值维度 | 具体描述 | 量化指标 |
|---------|---------|---------|
| **运营效率** | 统一管理界面，减少页面切换 | 操作时间减少 50%+ |
| **审核效率** | AI 辅助审核，批量操作 | 审核效率提升 70%+ |
| **数据安全** | 完善的权限控制和审计 | 100% 操作可追溯 |
| **决策支持** | 实时数据看板和报表 | 数据获取时间减少 80% |
| **财务透明** | 完整的交易记录和报表 | 财务对账效率提升 60% |

### 1.4 设计原则

```
1. 用户中心原则
   └── 以管理员工作流程为核心设计界面
   └── 减少操作步骤，提高效率

2. 数据驱动原则
   └── 关键数据可视化展示
   └── 支持数据导出和分析

3. 安全优先原则
   └── 完善的权限控制
   └── 敏感操作二次验证
   └── 完整的审计日志

4. 可扩展原则
   └── 模块化设计，支持功能扩展
   └── 灵活的权限配置

5. 一致性原则
   └── 统一的交互模式
   └── 一致的视觉风格
```

## 2. 功能模块

### 2.1 模块总览

```
AIAds 管理后台
├── 1. 数据总览 (Dashboard)
├── 2. 用户管理 (User Management)
├── 3. 广告主管理 (Advertiser Management)
├── 4. KOL 管理 (KOL Management)
├── 5. 活动管理 (Campaign Management)
├── 6. 订单管理 (Order Management)
├── 7. 财务管理 (Finance Management)
├── 8. 内容审核 (Content Moderation)
├── 9. 数据统计 (Analytics)
└── 10. 系统设置 (System Settings)
```

### 2.2 功能模块清单

| 模块编号 | 模块名称 | 优先级 | 核心功能 | 目标用户 |
|---------|---------|-------|---------|---------|
| **M01** | 数据总览 | P0 | 核心数据卡片、趋势图表、最近活动、待办事项 | 所有管理员 |

| **M05** | 活动管理 | P0 | 活动列表、活动详情、审核、状态管理、数据统计、异常监控 | 运营管理员、审核管理员 |
| **M06** | 订单管理 | P0 | 订单列表、订单详情、状态管理、纠纷处理、订单导出 | 运营管理员、财务管理员 |

### 2.3 功能优先级定义

| 优先级 | 定义 | 上线要求 |
|-------|------|---------|
| **P0** | 核心功能，必须上线 | MVP 必须包含 |
| **P1** | 重要功能，首月迭代 | 上线后 1 个月内 |
| **P2** | 优化功能，二期规划 | 上线后 3 个月内 |

```typescript
interface DashboardData {
  // 核心指标
  metrics: {
    totalUsers: number;          // 总用户数
    activeAdvertisers: number;   // 活跃广告主
    verifiedKols: number;        // 认证 KOL
    monthlyGmv: number;          // 本月 GMV
    monthlyRevenue: number;      // 本月收入

  // 趋势数据
  trends: {
    revenueTrend: TrendData[];   // 收入趋势
    userGrowthTrend: TrendData[]; // 用户增长趋势
  };

  // 最近活动
  recentActivities: Activity[];

  // 待办事项
  todos: TodoItem[];
}

interface TrendData {
  date: string;
  value: number;
  change?: number;  // 变化率
}

interface Activity {
  id: string;
  type: 'user' | 'kol' | 'order' | 'finance';
  description: string;
  timestamp: string;
  link?: string;
}

interface TodoItem {
  id: string;
  type: 'kol_review' | 'content_review' | 'withdrawal' | 'dispute';
  title: string;
  count: number;
  link: string;
  priority: 'high' | 'medium' | 'low';
}
```
