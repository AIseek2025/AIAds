## 1. 快速入门

### 1.1 系统访问

#### 访问方式

| 访问项 | 详情 |
|-------|------|
| **系统地址** | https://admin.aiads.com |
| **推荐浏览器** | Chrome 90+, Firefox 88+, Safari 14+, Edge 90+ |
| **推荐分辨率** | 1920x1080 或更高 |
| **网络要求** | 稳定的互联网连接 |

#### 系统要求

- **操作系统**: Windows 10+, macOS 10.15+, Linux
- **浏览器**: 支持 JavaScript 的现代浏览器
- **屏幕分辨率**: 最低 1366x768，推荐 1920x1080
- **网络带宽**: 最低 1Mbps，推荐 10Mbps+

### 1.2 账号登录

#### 首次登录

1. 访问 https://admin.aiads.com
2. 输入管理员分配的临时账号和密码
3. 首次登录会强制要求修改密码
4. 密码要求：
   - 至少 8 个字符
   - 包含大写字母
   - 包含小写字母
   - 包含数字
   - 包含特殊字符（!@#$%^&*等）

#### 日常登录

```
┌─────────────────────────────────────────┐
│         AIAds Admin Login               │
├─────────────────────────────────────────┤
│                                         │
│   邮箱：[________________]              │
│   密码：[________________]              │
│                                         │
│   ☐ 记住我（7 天内免登录）               │
│   ☐ 启用 MFA（推荐）                     │
│                                         │
│         [    登    录    ]              │
│                                         │
│   忘记密码？联系超级管理员重置           │
│                                         │
└─────────────────────────────────────────┘
```

**登录步骤**:
1. 输入注册邮箱
2. 输入密码
3. （可选）勾选"记住我"
4. （可选）启用 MFA 双因素认证
5. 点击"登录"按钮

#### MFA 双因素认证

**启用 MFA**:
1. 登录后进入"个人设置"
2. 点击"启用 MFA"
3. 使用手机扫描二维码
4. 输入验证码确认
5. 保存备用验证码

**登录时使用 MFA**:
1. 输入账号密码登录
2. 输入手机 APP 生成的 6 位验证码
3. 验证通过，登录成功

## 2. 功能详解

### 2.1 数据总览

#### 页面功能

数据总览是管理员的首页，展示平台核心运营数据。

#### 核心数据卡片

| 卡片名称 | 说明 | 点击跳转 |
|---------|------|---------|
| 总用户数 | 平台注册用户总数 | 用户管理 |
| 活跃广告主 | 近 30 天有活动的广告主 | 广告主管理 |
| 认证 KOL | 通过认证的 KOL 总数 | KOL 管理 |
| 本月 GMV | 本月成交订单总金额 | 订单管理 |
| 本月收入 | 本月平台佣金收入 | 财务管理 |
| 待审核 KOL | 等待审核的 KOL 数量 | KOL 审核 |
| 待审核内容 | 等待审核的内容数量 | 内容审核 |
| 待处理工单 | 等待处理的纠纷/提现 | 对应处理页面 |

#### 趋势图表

- **收入趋势**: 近 30 天收入变化折线图
- **用户增长**: 近 30 天用户增长折线图

**操作**:
- 点击右上角日期选择器切换时间范围
- 鼠标悬停查看具体数值
- 点击图例隐藏/显示数据系列

#### 最近活动

展示平台最近动态，包括：
- 用户注册/登录
- KOL 认证通过
- 活动创建/审核
- 订单状态变更

#### 待办事项

显示需要处理的待办任务：
- 待审核 KOL（点击直接跳转审核）
- 待审核内容（点击直接跳转审核）
- 待处理提现（点击直接跳转审核）
- 待处理纠纷（点击直接跳转处理）

### 2.2 用户管理

#### 用户列表

**列表字段**:
- 用户名
- 邮箱
- 手机号
- 用户类型（广告主/KOL/普通用户）
- 状态（正常/封禁）
- 注册时间
- 操作（查看/编辑/封禁）

**搜索功能**:
- 支持用户名、邮箱、手机号搜索
- 支持模糊匹配

**筛选功能**:
- 用户类型：全部/广告主/KOL/普通用户
- 状态：全部/正常/封禁
- 注册时间：自定义时间范围

#### 用户详情

**基本信息**:
- 头像
- 用户名
- 邮箱/手机
- 用户类型
- 账号状态
- 注册时间
- 最后登录时间

**行为数据**:
- 发起活动数
- 完成订单数
- 累计消费/收入
- 账户余额

**操作历史**:
- 登录记录
- 操作记录
- 状态变更记录

#### 用户操作

**编辑用户**:
1. 在用户列表点击"编辑"
2. 修改用户信息
3. 点击"保存"

**封禁用户**:
1. 在用户列表点击"封禁"
2. 选择封禁原因
3. 选择封禁时长（临时/永久）
4. 填写封禁说明
5. 二次验证（MFA/短信）
6. 确认封禁

**解封用户**:
1. 在用户列表点击"解封"
2. 填写解封说明
3. 确认解封

**重置密码**:
1. 在用户详情点击"重置密码"
2. 系统生成临时密码
3. 将临时密码发送给用户
4. 用户首次登录需修改密码

### 2.3 KOL 管理

#### KOL 列表

**列表字段**:
- 头像
- 名称
- 平台（TikTok/Instagram/YouTube 等）
- 平台账号
- 粉丝数
- 互动率
- 认证状态
- 评级
- 操作（查看/审核/编辑）

**筛选条件**:
- 平台：全部/TikTok/Instagram/YouTube/其他
- 粉丝数：全部/<1K/1K-5K/5K-1 万/1 万 -5 万/5 万 +
- 认证状态：全部/待审核/已认证/已拒绝
- 评级：全部/S/A/B/C

#### KOL 详情

**基本信息**:
- 头像和名称
- 平台账号链接
- 粉丝数、关注数、作品数
- 互动率（点赞 + 评论 + 分享）/ 粉丝数
- 认证状态和认证时间
- 评级和评级时间

**数据趋势**:
- 粉丝增长趋势（近 30 天）
- 互动率趋势（近 30 天）

**合作记录**:
- 进行中活动
- 已完成活动
- 合作收入统计

#### KOL 审核

**审核步骤**:
1. 进入"KOL 管理" → "待审核"
2. 查看 KOL 提交的认证资料
3. 点击 KOL 名称查看详情
4. 核实社交平台数据（可点击链接跳转核实）
5. 点击"通过"或"拒绝"
6. 填写审核意见（拒绝时必填）
7. 提交审核结果

**审核要点**:
- 粉丝数是否真实（检查增长曲线）
- 互动率是否正常（一般 1%-5%）
- 内容是否原创
- 是否有违规记录

#### 评级管理

**调整评级**:
1. 在 KOL 详情点击"调整评级"
2. 选择新评级（S/A/B/C）
3. 填写调整原因
4. 确认提交

**评级影响**:
- 影响 KOL 在列表中的排序
- 影响广告主选择 KOL 的优先级
- S 级 KOL 获得更多曝光机会

#### 黑名单管理

**加入黑名单**:
1. 在 KOL 详情点击"加入黑名单"
2. 选择加入原因
3. 填写说明
4. 确认提交

**黑名单效果**:
- KOL 无法接单
- KOL 不会在推荐列表中出现
- 通知 KOL 被封禁

### 2.4 活动管理

#### 活动列表

**列表字段**:
- 活动名称
- 广告主
- 活动类型
- 预算范围
- 参与 KOL 数
- 状态
- 创建时间
- 操作（查看/审核/编辑）

**筛选条件**:
- 状态：全部/待审核/进行中/已完成/已暂停/已拒绝
- 活动类型：全部/短视频/直播/图文/其他

#### 活动详情

**基本信息**:
- 活动名称和描述
- 广告主信息
- 活动类型和要求
- 预算和报价
- 活动时间
- 目标 KOL 类型

**参与 KOL**:
- 已报名 KOL 列表
- 已选中 KOL 列表
- KOL 任务完成情况

**数据统计**:
- 曝光量
- 点击量
- 转化数
- ROI

#### 活动审核

**审核步骤**:
1. 进入"活动管理" → "待审核"
2. 查看活动详情
3. 检查活动内容是否合规
4. 检查预算是否合理
5. 点击"通过"或"拒绝"
6. 填写审核意见（拒绝时必填）
7. 提交审核结果

### 2.5 订单管理

#### 订单列表

**列表字段**:
- 订单号
- 广告主
- KOL
- 关联活动
- 订单金额
- 状态
- 创建时间
- 操作（查看/管理）

**筛选条件**:
- 状态：全部/待付款/待接单/进行中/待审核/待发布/已完成/已取消/纠纷中
- 时间范围：自定义
- 广告主/KOL：搜索

#### 订单详情

**基本信息**:
- 订单号
- 广告主和 KOL 信息
- 关联活动
- 订单金额和平台佣金
- 创建时间

**任务要求**:
- 内容类型
- 内容要求
- 提交截止时间
- 发布时间要求

**进度跟踪**:
- 订单状态流转记录
- KOL 提交内容
- 审核记录
- 发布链接

#### 订单状态管理

**状态变更**:
1. 在订单详情点击"变更状态"
2. 选择新状态
3. 填写变更原因（如需要）
4. 确认提交

**常见状态变更**:
- 待付款 → 已付款（广告主付款后自动）
- 待接单 → 进行中（KOL 接单后自动）
- 进行中 → 待审核（KOL 提交内容后自动）
- 待审核 → 待发布（审核通过后自动）
- 待发布 → 已完成（KOL 发布后自动）

#### 纠纷处理

**处理步骤**:
1. 在订单列表筛选"纠纷中"状态
2. 点击进入纠纷订单详情
3. 查看纠纷详情和双方陈述
4. 查看相关证据（聊天记录、内容截图等）
5. 做出裁决（支持广告主/支持 KOL/折中方案）
6. 填写裁决说明
7. 提交裁决结果

### 2.6 财务管理

#### 财务总览

**核心指标**:
- 本月收入
- 待审核提现笔数和金额
- 待确认充值笔数和金额
- 发票待处理数量

#### 充值管理

**充值列表**:
- 充值单号
- 广告主
- 充值金额
- 支付方式
- 状态（待确认/已确认/已取消）
- 充值时间
- 操作（确认/查看详情）

**确认充值**:
1. 查看充值申请详情
2. 核实支付凭证
3. 点击"确认"
4. 充值金额计入广告主账户

#### 提现审核

**提现列表**:
- 提现单号
- KOL
- 提现金额
- 收款账户
- 状态（待审核/审核通过/已打款/已拒绝）
- 申请时间
- 操作（审核/查看详情）

**审核步骤**:
1. 查看提现申请详情
2. 核实 KOL 账户余额
3. 检查是否有异常交易
4. 点击"通过"或"拒绝"
5. 填写审核意见（拒绝时必填）
6. 提交审核结果
7. （通过后）财务打款并确认

**大额提现**:
- 单笔超过¥10,000 需要二次验证
- 可能需要额外审核材料

#### 交易记录

**交易列表**:
- 交易时间
- 交易类型（充值/提现/消费/收入/退款）
- 用户
- 金额（+ 收入/-支出）
- 状态
- 关联订单/活动
- 操作（查看详情）

**搜索筛选**:
- 交易类型
- 时间范围
- 用户搜索
- 金额范围

**导出交易记录**:
1. 设置筛选条件
2. 点击"导出"按钮
3. 选择导出格式（Excel/CSV）
4. 下载导出文件

#### 财务报表

**报表类型**:
- 日报：每日收入支出汇总
- 周报：每周收入支出汇总
- 月报：每月收入支出汇总
- 自定义报表：自定义时间范围

**生成报表**:
1. 进入"财务管理" → "财务报表"
2. 选择报表类型
3. 选择时间范围
4. 点击"生成报表"
5. 查看或下载报表

### 2.7 内容审核

#### 待审核内容

**内容列表**:
- 内容缩略图
- 内容标题/描述
- 提交 KOL
- 关联活动/订单
- 内容类型（视频/图片/文字）
- 提交时间
- 操作（审核/查看详情）

#### 内容详情

**内容展示**:
- 视频播放器/图片预览/文字内容
- 内容描述和标签
- 提交 KOL 信息
- 关联订单信息

**审核操作**:
1. 查看完整内容
2. 检查内容合规性
3. 点击"通过"或"拒绝"
4. 填写审核意见（拒绝时必填）
5. 提交审核结果

**审核要点**:
- 内容是否符合法律法规
- 内容是否清晰完整
- 是否符合品牌要求
- 是否为原创内容
- 是否有敏感词或违规内容

#### 敏感词管理

**查看敏感词**:
1. 进入"内容审核" → "敏感词管理"
2. 查看当前敏感词列表

**添加敏感词**:
1. 点击"添加敏感词"
2. 输入敏感词
3. 选择敏感级别（高/中/低）
4. 填写说明
5. 确认添加

**删除敏感词**:
1. 在敏感词列表找到目标词
2. 点击"删除"
3. 确认删除

## 3. 最佳实践

### 3.1 使用技巧

#### 快捷键

| 快捷键 | 功能 |
|-------|------|
| `Ctrl/Cmd + K` | 全局搜索 |
| `Ctrl/Cmd + /` | 打开帮助 |
| `Esc` | 关闭弹窗/返回上级 |
| `Ctrl/Cmd + S` | 保存当前编辑 |
| `F5` | 刷新当前页面 |

#### 批量操作

**批量审核 KOL**:
1. 在 KOL 列表勾选多个 KOL
2. 点击顶部"批量审核"按钮
3. 选择审核结果（通过/拒绝）
4. 填写审核意见
5. 确认提交

**批量审核内容**:
1. 在待审核内容列表勾选多个内容
2. 点击顶部"批量审核"按钮
3. 选择审核结果
4. 填写审核意见
5. 确认提交

#### 数据导出

**导出最佳实践**:
- 单次导出数据量建议不超过 10 万条
- 大数据量导出建议使用时间范围筛选
- 导出文件注意妥善保管
- 敏感数据导出后及时删除

### 3.2 效率提升

#### 待办事项处理

**每日工作流程**:
1. 上班后首先查看数据总览
2. 处理待办事项列表中的任务
3. 优先处理高优先级任务
4. 定期刷新查看新待办

**待办优先级**:
- **高优先级**: 纠纷处理、大额提现、紧急审核
- **中优先级**: 常规审核、用户问题
- **低优先级**: 数据导出、报表生成

#### 搜索技巧

**精确搜索**:
- 使用完整邮箱/手机号搜索用户
- 使用订单号精确查找订单

**模糊搜索**:
- 使用用户名/昵称部分字符搜索
- 使用 KOL 平台账号部分字符搜索

**组合筛选**:
- 结合多个筛选条件缩小范围
- 保存常用筛选条件

#### 通知管理

**通知设置**:
1. 进入"个人设置" → "通知设置"
2. 选择需要接收的通知类型
3. 选择通知方式（站内信/邮件/短信）
4. 保存设置

**建议开启的通知**:
- 审核任务通知
- 纠纷处理通知
- 系统公告通知

### 3.3 安全建议

#### 账号安全

- 定期修改密码（建议每 90 天）
- 启用 MFA 双因素认证
- 不在公共电脑登录
- 不与他人共享账号

#### 操作安全

- 敏感操作前仔细核对
- 封禁/删除等操作需二次确认
- 大额资金操作需二次验证
- 定期查看操作日志

#### 数据安全

- 不随意导出用户数据
- 导出的敏感数据妥善保管
- 不再需要的数据及时删除
- 发现数据异常及时上报

## 4. 常见问题 FAQ

### 4.1 登录问题

**Q: 忘记密码怎么办？**
- A: 点击登录页面的"忘记密码"链接，输入注册邮箱，系统将发送重置密码邮件。如未收到邮件，请联系超级管理员。

**Q: MFA 验证码收不到怎么办？**
- A: 
  1. 检查手机信号和网络连接
  2. 检查短信是否被拦截
  3. 尝试重新发送验证码
  4. 如仍收不到，联系超级管理员暂时关闭 MFA

**Q: 账号被禁用怎么办？**
- A: 账号被禁用通常是因为违规操作。请联系您的直属主管或超级管理员，说明情况申请解封。

**Q: 登录后提示"无权限访问"怎么办？**
- A: 您的账号可能没有被分配相应角色的权限。请联系超级管理员为您配置权限。

### 4.2 功能使用问题

**Q: 如何批量审核 KOL？**
- A: 
  1. 进入"KOL 管理" → "待审核"
  2. 勾选需要审核的多个 KOL
  3. 点击顶部"批量审核"按钮
  4. 选择审核结果（通过/拒绝）
  5. 填写审核意见
  6. 确认提交

**Q: 数据导出失败怎么办？**
- A: 
  1. 检查数据量是否过大（建议单次不超过 10 万条）
  2. 尝试缩小时间范围
  3. 检查网络连接
  4. 如仍失败，联系技术支持

**Q: 审核误操作怎么办？**
- A: 
  1. 立即停止后续操作
  2. 联系超级管理员说明情况
  3. 查看审计日志确认操作记录
  4. 根据超级管理员指导进行修正

**Q: 如何查看某个用户的操作历史？**
- A: 
  1. 进入"用户管理"
  2. 找到目标用户并点击"详情"
  3. 在用户详情页查看"操作历史"标签
  4. 也可在"系统设置" → "操作日志"中搜索该用户

### 4.3 系统问题

**Q: 页面加载很慢怎么办？**
- A: 
  1. 检查网络连接
  2. 清除浏览器缓存
  3. 尝试使用其他浏览器
  4. 如持续缓慢，联系技术支持

**Q: 系统提示"服务器错误"怎么办？**
- A: 
  1. 记录错误信息和时间
  2. 刷新页面重试
  3. 如持续报错，联系技术支持并提供错误信息

**Q: 数据不一致怎么办？**
- A: 
  1. 刷新页面
  2. 清除缓存后重试
  3. 如仍不一致，联系技术支持排查

**Q: 系统维护时间是什么时候？**
- A: 系统定期维护时间为每周日凌晨 02:00-06:00（UTC+8）。维护期间可能无法访问，请提前安排工作。

### 4.4 权限问题

**Q: 为什么看不到某个功能菜单？**
- A: 您的角色可能没有该功能的访问权限。请联系超级管理员为您配置权限。

**Q: 为什么操作按钮是灰色的？**
- A: 您有查看权限但没有操作权限。需要向超级管理员申请相应操作权限。

**Q: 如何申请更多权限？**
- A: 
  1. 向直属主管说明需要权限的原因
  2. 主管审批后通知超级管理员
  3. 超级管理员配置权限后生效

**Q: 权限配置后多久生效？**
- A: 权限配置后立即生效。如未生效，请退出重新登录。

### 4.5 财务问题

**Q: 提现审核的时效要求是什么？**
- A: 提现审核需在 24 小时内完成。大额提现（>¥10,000）可能需要额外审核时间。

**Q: 充值未到账怎么办？**
- A: 
  1. 检查充值状态是否为"待确认"
  2. 核实支付凭证是否上传
  3. 联系财务管理员确认
  4. 如已确认但未到账，联系技术支持

**Q: 财务报表如何生成？**
- A: 
  1. 进入"财务管理" → "财务报表"
  2. 选择报表类型（日报/周报/月报/自定义）
  3. 选择时间范围
  4. 点击"生成报表"
  5. 查看或下载报表
