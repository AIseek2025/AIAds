## 五、模板使用说明

### 5.1 变量说明

| 变量 | 说明 | 示例 |
|-----|------|------|
| [姓名] | 收件人姓名 | 张三 |
| [公司] | 公司名称 | XX 科技 |
| [CODE] | 邀请码 | ADV-001-001 |
| [URL] | 注册链接 | https://aiads.com/register |
| [平台] | 社交平台 | TikTok |
| [名字] | 发送人姓名 | 小李 |
| [PHONE] | 联系电话 | 400-XXX-XXXX |
| [DASHBOARD_URL] | 数据看板链接 | https://aiads.com/dashboard |

### 5.2 使用建议

1. **个性化**：尽量使用收件人姓名，避免群发感
2. **时机选择**：工作日 10-11 点、14-16 点发送效果最佳
3. **跟进节奏**：首次发送后 3 天、7 天各跟进一次
4. **A/B 测试**：不同模板小批量测试，选择最优版本
5. **渠道适配**：根据渠道特点调整语气和格式
