# AIAds 种子用户招募计划

## 一、招募目标

| 用户类型 | 目标数量 | 完成时间 | 负责人 |
|---------|---------|---------|--------|
| 广告主 | 500 家 | 2 天 | 运营经理 |
| KOL | 10,000 名 | 2 天 | 运营经理 |

## 二、目标用户画像

### 2.1 广告主画像

```
- 类型：工厂/跨境电商/品牌方
- 地区：广东/浙江/江苏/福建
- 年营收：1000 万 -1 亿
- 海外营销预算：10 万 -100 万/年
- 痛点：成本高、找不到合适 KOL、操作复杂
```

**详细特征**:
- 行业分布：电子产品 30%、服装服饰 25%、家居用品 20%、美妆个护 15%、其他 10%
- 企业规模：50-500 人
- 决策人：营销总监/创始人/跨境电商负责人
- 现有营销渠道：Amazon、AliExpress、独立站

### 2.2 KOL 画像

```
- 平台：TikTok/YouTube/Instagram
- 粉丝：1,000-50,000
- 地区：美国/欧洲/东南亚
- 类别：时尚/美妆/科技/生活
- 痛点：粉丝少接不到广告、收入不稳定
```

**详细特征**:
- 平台分布：TikTok 50%、Instagram 30%、YouTube 20%
- 地区分布：美国 40%、欧洲 30%、东南亚 20%、其他 10%
- 内容类别：时尚 25%、美妆 25%、科技 20%、生活 20%、其他 10%
- 互动率要求：≥2%
- 更新频率：月更≥4 条

## 四、邀请码系统

### 4.1 邀请码规则

**格式**: `[类型]-[渠道]-[序号]`

**示例**:
- `ADV-001-001`：广告主 - 渠道 001-第 1 个
- `KOL-TIK-0001`：KOL-TikTok-第 1 个

### 4.2 渠道编码

| 编码 | 渠道名称 | 适用类型 |
|-----|---------|---------|
| 001 | 行业协会 | 广告主 |
| 002 | 跨境电商群 | 广告主 |
| 003 | 行业媒体 | 广告主 |
| 004 | 线下展会 | 广告主 |
| 005 | 供应商推荐 | 广告主 |
| TIK | TikTok | KOL |
| INS | Instagram | KOL |
| YT | YouTube | KOL |
| MCN | MCN 机构 | KOL |
| REF | 推荐邀请 | 通用 |

### 4.3 邀请码权益

**广告主权益**:
- 首单免费（最高¥5,000）
- 专属客服 1v1 指导
- 优先体验新功能
- 推荐奖励：每成功推荐 1 人奖励¥100

**KOL 权益**:
- 优先派单
- 10% 佣金加成
- 快速提现（T+1 到账）
- 推荐奖励：每成功推荐 1 人奖励¥100
