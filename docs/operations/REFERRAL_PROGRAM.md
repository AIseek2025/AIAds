# AIAds 推荐奖励计划

## 一、计划概述

**目标**：通过用户推荐实现裂变式增长，降低获客成本，提高用户质量。

**核心机制**：老用户邀请新用户，双方获得奖励。

## 四、奖励发放流程

### 4.1 发放条件验证

```
广告主推荐验证流程：
1. 被推荐人完成注册
2. 完成企业资质验证
3. 完成首次充值（≥¥1,000）
4. 系统自动发放奖励至推荐人账户

KOL 推荐验证流程：
1. 被推荐人完成注册
2. 完成账号验证（粉丝数、互动率）
3. 完成首单投放
4. 系统自动发放奖励至推荐人账户
```

## 五、防作弊机制

### 5.1 作弊行为定义

以下行为视为作弊：
- 同一用户多账号注册
- 虚假企业资质
- 刷量行为（粉丝、互动）
- 恶意套现
- 其他违反平台规则的行为

### 5.2 检测措施

| 检测项 | 方法 | 阈值 |
|-------|------|------|
| 设备指纹 | Device ID 检测 | 同设备≤3 账号 |
| IP 检测 | IP 地址分析 | 同 IP≤5 账号/天 |
| 手机验证 | 短信验证码 | 一手机号一账号 |
| 企业验证 | 营业执照核验 | 一执照一账号 |
| 行为分析 | 异常操作检测 | 人工审核 |

### 5.3 处罚措施

| 违规程度 | 处罚措施 |
|---------|---------|
| 轻度 | 警告 + 冻结奖励 |
| 中度 | 封禁账号 + 没收奖励 |
| 重度 | 永久封禁 + 法律追责 |

## 八、常见问题

### Q1：推荐奖励何时到账？
A：被推荐人完成验证后 1-3 个工作日内到账。

### Q2：奖励可以提现吗？
A：现金奖励可以提现，最低¥50 起提。

### Q3：有推荐人数上限吗？
A：没有上限，推荐越多奖励越多，还有阶梯 bonus！

### Q4：被推荐人必须是新用户吗？
A：是的，必须是首次注册 AIAds 的新用户。

### Q5：如何查看我的推荐记录？
A：登录后台，在「我的推荐」页面查看完整记录。

### Q6：推荐奖励有有效期吗？
A：奖励到账后 180 天内有效，请及时使用或提现。
